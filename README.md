# Neovim config

## My personal configurations for Neovim

### To install

```
rm -rf $HOME/.local/share/nvim

rm -rf $HOME/.local/state/nvim

rm -rf $HOME/.config/nvim

git clone https://github.com/kernpinoy/nvim-config $HOME/.config/nvim

cd nvim-config

chmod +x setup-nvim.sh

./setup-nvim.sh
```
